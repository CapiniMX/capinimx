INSTRUCCIONES DE DEPLOY

OPCIÓN 1: Reemplazo Directo (Recomendado)
  1. Navegar a: /contenido_web/aprendiendo-luk/
  2. Copiar: cp index_v2.0.html index.html
  3. Verificar en navegador
  4. Listo ✅

OPCIÓN 2: Mantener Ambas Versiones
  1. Mantener index.html (v1.0 actual)

  4. Listo ✅

OPCIÓN 3: Testing Local
  1. cd /contenido_web/aprendiendo-luk/
  2. python3 -m http.server 8000
  3. Abrir: http://localhost:8000/index_v2.0.html
  4. Testear completamente
  5. Deploy cuando confirme ✅
