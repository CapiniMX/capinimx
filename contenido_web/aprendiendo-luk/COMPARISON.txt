7. ESPACIADO - SISTEMA DE VARIABLES
─────────────────────────────────────────────────────────────────────────

v1.0
  Espaciado:  Hardcodeado (12px, 16px, 20px, etc.)
  Sistema:    Sin variables, valores dispersos
  
  ⚠️ Difícil de mantener, inconsistencias

v2.0
  Espaciado:  Variables CSS definidas
  Sistema:
    --spacing-xs:   4px
    --spacing-sm:   8px
    --spacing-md:   16px (estándar)
    --spacing-lg:   24px (secciones)
    --spacing-xl:   32px (header/footer)
  
  ✅ Consistente, fácil de cambiar, profesional

VENTAJAS:
  • Fácil auditar y modificar
  • Consistencia garantizada
  • Mejor para mantener en el futuro

9. TRANSICIONES Y ANIMACIONES
─────────────────────────────────────────────────────────────────────────

v1.0
  Transiciones:     0.3s ease (básico)
  Easing:           Linear principalmente
  Animaciones:      Simples (scale, color)
  Duración:         Inconsistente (0.3s, 0.5s, 0.9s)
  
  ⚠️ Funcionales pero robóticas

v2.0
  Transiciones:     0.3s cubic-bezier(0.4, 0, 0.2, 1)
  Easing:           Professional curves
  Animaciones:
    • Elevación: translateY(-2px a -4px)
    • Scale: 0.95-1
    • Opacity: 0-1
    • Shine: left -100% → 100%
  Duración:         Consistente (0.3s estándar, 0.15s rápido)
  
  ✅ Smooth, natural, profesional

RESULTADO PERCEPTUAL:
  v2.0 se "siente" significativamente más fluido
  y responsivo que v1.0

10. RESPONSIVE DESIGN
─────────────────────────────────────────────────────────────────────────

v1.0 Breakpoints:
  @media (max-width: 480px)
    • Operation display: 2.5em
    • Color grid: 2x2 (ambiguo)
  
  ⚠️ Limitado a un breakpoint
  ⚠️ Tablet (768px) no optimizado

v2.0 Breakpoints:
  @media (max-width: 768px)
    • Ajustes menores
  
  @media (max-width: 480px)
    • Header adapta con wrap
    • Tabs pasan a segundo row
    • Operation: 2.5em
    • Color grid: 2x2
    • Padding: 16px
  
  @media (max-width: 360px)
    • Operation: 2em (más pequeño)
    • Result score: 42px
    • Optimización extra para móvil pequeño
  
  ✅ Múltiples breakpoints, mejor cobertura

RESULTADO:
  • Mejor en mobile (320px)
  • Mejor en mobile medio (480px)
  • Mejor en mobile pequeño (360px)
  • Mejor en tablet (sin quebrar layout)
