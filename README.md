## Qué es este repositorio

Este es el repositorio de **identidad de marca** de CAPINI. Contiene toda la información extraída y consolidada de nuestra página web, redes sociales y materiales existentes:

- **Marca** — nombre, significado, tagline, logo
- **Colores** — paleta primaria y secundaria con códigos hex
- **Tipografía** — fuentes del logo y recomendadas para materiales
- **Voz y tono** — cómo se comunica CAPINI (profesional, cálida, empática, accesible)
- **Copy** — textos clave, frases, headlines
- **Estilo visual** — lineamientos de diseño para cualquier pieza gráfica
- **Audiencias** — segmentos de público por edad y necesidad
- **Catálogo de servicios** — evaluaciones, terapias, talleres y programas
- **Diferenciadores** — qué hace único a CAPINI

Todo esto sirve como referencia para generar contenido, diseños, campañas y materiales de comunicación manteniendo la marca consistente.
